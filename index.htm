Configuration

  • git --version

  • git --help

  • git config

    # shows full configuration
    git config --list

    # shows user name
    git config user.name
Projects

  • git init
    Initialize new git repository

    # create a new directory and initialize it with git init function
    git init my-new-repo

  • git status
    Shows the state of the working directory

  • git add
    Stages a change

    # create first file and stage it
    touch README.md
    git add README.md

  • git commit
    Saves the snapshot to the project history

    # take a snapshot of the stagin aera
    git commit -m "add README to initial commit"
Remote

  • git remote
    Shows the shortnames of each remote handle you've specified

    # show the URLs for remotes
    git remote -v

  • git remote add
    Adds the new remote repository

    # adds remote repo, commit and push files
    git remote add origin git@gitlab.com:xxx/my-repo.git
    git add .
    git commit -m "Initial commit"
    git push -u origin master

  • git fetch
    Fetch all files from remote repo that don't exist in local repository

    git fetch origin

  • git pull
    Fetch and merge

    git pull

  • git push

    git push origin master
Branches

  • git branch
    Shows the branches being worked on locally
